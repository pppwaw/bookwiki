# Search Basics

Graph search keeps a <ConceptLink slug="frontier">frontier</ConceptLink> of generated nodes that are waiting to be expanded. The exact data structure depends on the search strategy. <SourceRef id="demo-textbook-p001" quote="The frontier stores generated but unexpanded nodes." />
