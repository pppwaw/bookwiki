# Heuristics

A heuristic estimates remaining cost from a state to a goal. An <ConceptLink slug="admissible-heuristic">admissible heuristic</ConceptLink> never overestimates that remaining cost. <SourceRef id="demo-textbook-p010" quote="An admissible heuristic is optimistic." />

## A Star

A star search orders the <ConceptLink slug="frontier">frontier</ConceptLink> by combining path cost so far with estimated remaining cost. <SourceRef id="demo-lecture-slide05" quote="A star uses g plus h to rank candidates." />
